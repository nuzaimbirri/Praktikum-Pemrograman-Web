## Instalasi dan Penggunaan

Karena ini adalah website statis, tidak diperlukan proses instalasi yang kompleks.

1.  *Clone repository ini:*
    bash
    git clone https://github.com/nuzaimbirri/Praktikum-Pemrograman-Web.git 
    
2.  *Masuk ke direktori proyek:*
    bash
    cd Tugas Akhir Percobaan 2
    
3.  **Buka file index.html:**
    Buka index.html di browser pilihan kamu (misalnya Google Chrome, Firefox, atau Edge) untuk melihat website secara lokal.

## Struktur file penting

- `index.html` — halaman utama proyek (konten dan struktur).
- `style.css` — styling utama (tema, layout, responsif).
- `foto/` — folder aset gambar (foto profil, logo library yang digunakan).
- `screenshoot/` — folder tempat menyimpan screenshoot.
- `README.md` — dokumentasi ini.
